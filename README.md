# APIREST MEN

- This is a small APIREST with CRUD operations, you can try it with Posman.

### Used technologies:
 
- Node.js
- Express
- MongoDB (mongoose)
- Nodemon
#### Packages:

*    cors
*    dotenv
*    express
*    express-validator
*    mongoose
*    method-override
*    body-parser

### Commands:

```
npm start
npm run dev
```
